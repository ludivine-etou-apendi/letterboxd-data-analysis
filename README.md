Projet Letterboxd Python 
🎥 Projet Web Scraping & Analyse de Films (Letterboxd)

Ce projet Python vise à scraper des données détaillées (Note, Genre, Réalisateurs etc.) sur 10 795 films depuis Letterboxd.

Pour executer le scraping tapez "python3 scraperletterboxd.py"
⚠️ ATTENTION : L'étape de scraping des 10 795 films individuels (scraperletterboxd.py) nous a pris plus de 16 heures.

Pour exécuter le script sur un échantillon de liens au lieu de la totalité (pour des tests), vous devez modifier la variable SCRAPING_LIMIT dans la section Configuration du code. Vous pouvez y renseigner le nombre d'URLs que vous souhaitez traiter.
