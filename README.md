Projet Letterboxd Python 
🎥 Projet Web Scraping & Analyse de Films (Letterboxd)

Ce projet Python vise à scraper des données détaillées (Note, Genre, Réalisateurs etc.) sur 10 795 films depuis Letterboxd.
⚠️ ATTENTION : L'étape de scraping des films individuels (scraperletterboxd.py) nous a pris plus de 16 heures.
Le pipeline inclut l'extraction des URL, le scraping des détails, le nettoyage et l'exportation en CSV/JSON.
L'objectif est d'analyser les tendances cinématographiques (impact des festivals, évolution de la popularité etc.).
